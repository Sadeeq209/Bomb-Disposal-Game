Bomb Disposal Game

A text-based bomb defusal puzzle game written in C++. Players take on the role of a bomb disposal officer, investigating crime scenes, analyzing evidence, solving logical puzzles, and defusing explosive devices before time runs out.

Features

- Story-driven bomb disposal missions
- Investigation-based gameplay
- Logic puzzles requiring deduction rather than guessing
- Unique PIN and wire solutions for every case
- Player profile system
- Mission progression
- Modular object-oriented design

File Responsibilities

- main.cpp - Starts the game and controls the main program flow.
- bomb.h / bomb.cpp - Bomb mechanics, countdown timer, PIN verification, wire cutting, and defusal logic.
- mission.h / mission.cpp - Mission stories, police briefings, investigations, clues, and mission progression.
- profile.h / profile.cpp - Player profiles, save/load functionality, and mission completion tracking.
- utils.h / utils.cpp - Shared helper functions used throughout the project.
